# chip8

This is a chip8 emulator I wrote in rust.


## How to use

1. Install [Rust](https://rustup.rs/)

2. Open this directory

3. Run the command `cargo run --release -- path/to/rom.ch8`  
(It may take a while to compile for the first time)
